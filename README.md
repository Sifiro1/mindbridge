# MindBridge — EdTech AI Platform

MindBridge помогает студентам раскрывать потенциал через игру и искусственный интеллект.
Система анализирует поведение, прогнозирует результаты и помогает преподавателям понимать реальный потенциал студентов.

## Технологический стек
- **Backend:** Go (Gin/Fiber, PostgreSQL, Redis)
- **AI Layer:** Python (FastAPI, scikit-learn, transformers)
- **Frontend:** React / Next.js
- **Messaging:** Kafka / NATS
- **Storage:** S3 / MinIO
- **Monitoring:** Prometheus + Grafana
